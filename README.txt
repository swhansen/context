LightRod README.txt
-------------------

To install a basic location server follow the instructions after running index.php 
in your browser, assuming you have unpacked this into your web path.  If not, copy
this directory tree into your web path.

This package also consists of the LightRod Open Augmented Reality (OAR) server and
the LightRod location browser, in /oar-server and /oar-browser respectively.  


Further instructions on using these packages are available at

http://www.lightrod.org



Developer support is available on the forum from the website above, or from 
peter AT lightrod.org.
